<h1>Introduction</h1>
In the Railway reservation system, we have tried to develop a platform to make the reservation system more efficient, easier and fast. It explains the systematic procedure of railway reservation in India. This project is developed using the C language.This railway reservation system is beneficial to all the user who have to book the ticket online instead of standing in the long queues at the reservation counter.

<h1>Features</h1>
<ul>
  <li>Reserve a Ticket</li>
  <li>View All Available Trains</li>
  <li>Cancel Reservation</li>
  <li>View Ticket</li>
</ul>

<h1>4W's and 1H's</h1>
<h3>Who?</h3>
The Program can be used by any valid user who needs to reserve a ticket in train.
<h3>What?</h3>
It can be used to book train as well as cancel train.
<h3>When?</h3>
The program will be used when user needs to travel, thus book new ticket.
<h3>Where?</h3>
The program can be run on any basic computer that most users have these days.
<h3>How?</h3>
The user is presented with multiple options for booking and cancellation of tickets.
